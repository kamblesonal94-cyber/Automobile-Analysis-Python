# Automobile Data Analysis

## Project Overview
This project analyzes automobile data using Python to identify patterns, trends, and factors influencing vehicle prices.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Clean and preprocess automobile data
- Perform exploratory data analysis (EDA)
- Visualize key trends and relationships
- Generate actionable business insights

## Key Insights
- Engine size and horsepower show a positive relationship with vehicle price.
- Premium automobile brands generally have higher average prices.
- Fuel type and vehicle specifications impact pricing trends.
